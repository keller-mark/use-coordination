# use-coordination

A library for coordinated multiple views in React-based visualization systems.


## Usage

```sh
npm install use-coordination
```

## Development

Install pnpm v8

### Setup

```sh
git clone 
pnpm install
```

### Run demo

```sh
pnpm start
```

### Lint and format

```sh
pnpm lint
pnpm format
```

### Build library

```sh
pnpm build
```

### Build demo

```sh
pnpm build-demo
```

### Monorepo tasks

```sh
pnpm meta-updater
```
